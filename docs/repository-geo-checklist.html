Repository GEO checklist

A repository is easier to discover when the homepage, metadata, docs, and machine-readable files all point in the same direction.

Home GitHub

Must-have items

  • clear one-line repository description
  • filled topics and homepage URL
  • README with problem-shaped headings
  • `llms.txt`, `AGENTS.md`, and `CITATION.cff`
  • GitHub Pages landing pages
  • `robots.txt` and `sitemap.xml`
