# 14-Day Discovery Loop

## Day 1

- Publish the repository
- Enable Pages
- Fill description, homepage, topics

## Day 2-3

- Publish one X thread
- Publish one external article
- Link both back to the docs site

## Day 4-7

- Watch repository traffic in GitHub Insights
- Note which pages get views
- Tighten titles and opening paragraphs

## Day 8-10

- Add one more keyword landing page
- Improve anchor text from profile README or external pages

## Day 11-14

- Review clones, visitors, and referring sites
- Rewrite the weakest-performing page
- Double down on the keyword cluster that is actually getting discovered
